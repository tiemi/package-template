# processador_imagem

Description. 
The package processador_imagem is used to:
	Processing:
		- Histogram matching
		- Structural similarity
		-Resize image
	Utils:
		-Read image
		-Save image
		-Plot image
		-Plot result
		-Plot histogram

O pacote package processador_imagem é usado para:
	Processing:
		- Correspondência de histograma
		- Semelhança estrutural
		-Redimensionar imagem
	Utils:
		-Ler imagens
		-Gravar imagens
		-Plot imagem
		-Plot resultado
		-Plot histograma

## Installation

pip install processador_imagem
